# TripPlanner-AI-Multi-Agent-Travel-Planner


# How to run
1. Create the Virtual Enivornmnet

```bash
conda create -n travel python=3.11 -y
```

2. Activate the Enivornmnet
```bash
conda activate travel
```

3. Install the requirements

```bash
pip install -r requirements.txt
```
4. Running the application
```bash
 python app.py
```
